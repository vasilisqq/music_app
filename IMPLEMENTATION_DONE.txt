🎯 КЛЮЧЕВЫЕ ИЗМЕНЕНИЯ
═════════════════════

1. ТАЙМАУТЫ (10 СЕКУНД):
   ─────────────────────
   - Константа: REQUEST_TIMEOUT_MS = 10000 мс
   - На каждом request: setTransferTimeoutAttribute(REQUEST_TIMEOUT_MS)
   - QTimer срабатывает → reply.abort() → отправка сигнала об ошибке
   - Автоматическая очистка таймеров после завершения

2. СТРУКТУРИРОВАННАЯ ОБРАБОТКА ОШИБОК:
   ────────────────────────────────────
   
   def method():
       TRY: подготовка запроса + отправка
       EXCEPT Exception: emit error_signal
   
   def _on_reply():
       TRY:
           IF reply.error() == NoError:
               TRY: JSON парсинг
               EXCEPT JSONDecodeError: emit error
           ELSE:
               call _handle_error()
       EXCEPT Exception: emit critical_error
       FINALLY: reply.deleteLater() + cleanup_timeout()

3. МЕТОДЫ УПРАВЛЕНИЯ ТАЙМАУТАМИ:
   ───────────────────────────────
   _setup_timeout(reply, msg):
       - Создаёт QTimer с timeout_msg
       - Сохраняет в словарь self.timeout_timers
       - На таймаут → reply.abort() + emit error
   
   _cleanup_timeout(reply):
       - Останавливает таймер
       - Удаляет из словаря

4. УЛУЧШЕННАЯ ОБРАБОТКА ОШИБОК:
   ────────────────────────────────
   - Специфичная обработка json.JSONDecodeError
   - Специфичная обработка ValueError (Pydantic)
   - Fallback обработка с errors="ignore"
   - Логирование в консоль (print)
   - Информативные сообщения пользователю
